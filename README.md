# GraphQL Movie

Movie App built with React, Apollo and GraphQL  
used **"React"**, **"GraphQL"**, **"Apollo"**, **"React Router"**.

</br>

## GOAL

- Understand the basics of graphQL and Apollo.
- Implement services from back-end to front-end through GraphQL.

</br>

## Feature List

- Replace Redux
- Replace Fetch
- Get Movies
- See Movie
- Add to Favourites
